# College_chatbot


# Description:
A smart chatbot designed for college students and staff, providing information about courses, events, campus facilities, administrative processes, exams, and more. It integrates real-time updates from the college website and internal databases.

# Features:

Provides instant answers to queries on courses, events, hostel menus, and exam prep

PDF downloads of previous year question papers

Uses AI/NLP to understand and respond naturally

# Technologies Used:

Python (FastAPI, LangChain, OpenAI APIs)

Web scraping (Crawl4AI)

Database: MySQL / SQLite

Frontend integration via APIs

# Usage:

Students can query the bot for information

Staff can update schedules and events

Can be deployed on a college intranet or public-facing portal

Can be generlized project to get urls of website and get doucment to create custom chatbot
